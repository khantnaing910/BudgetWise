# BudgetWise 💸

BudgetWiseは、ピュアなHTML、CSS、JavaScriptで構築された、洗練されたモダンなクライアントサイドの個人資産管理（家計簿）アプリです。すべてブラウザ上で、収入の記録、支出の管理、予算の設定、貯金目標のモニタリングを行うことができます。

![BudgetWise Preview](https://via.placeholder.com/800x400.png?text=BudgetWise+Dashboard)

## 🌟 機能

- **ダッシュボード:** 純資産、貯蓄率、最近の取引を俯瞰して確認できます。
- **取引管理:** 直感的なインターフェースで収入と支出を記録・分類します。
- **予算プランナー:** カテゴリ（食費、家賃、交通費など）ごとに支出上限を設定し、進捗を視覚的に追跡します。
- **貯金目標:** 財務的な目標を設定し、プログレスバーが満たされていくのを確認できます。
- **インタラクティブなチャート:** Chart.jsを活用した美しい月間棒グラフやカテゴリ別のドーナツグラフ。
- **レポートとエクスポート:** 月ごとの概要の確認、CSV形式での取引データのエクスポート、見やすいPDFレポートの印刷が可能です。
- **ダークモード:** 夜間の使用にも優しい、細部までこだわったダークテーマ。
- **プライバシーファースト:** 100%クライアントサイドで動作します。財務データがデバイスの外部に送信されることはなく、ブラウザの `localStorage` に安全に保存されます。

## 🛠️ インストールと使い方

BudgetWiseは静的なアプリケーションであるため、ビルドツールやサーバーは不要です。

1. リポジトリをクローンまたはダウンロードします:
   ```bash
   git clone https://github.com/khantnaing910/BudgetWise.git
   ```
2. フォルダを開き、`index.html` をダブルクリックして最新のブラウザで開きます。
3. 記録を始めましょう！ウェルカム画面の「Load Sample Data（サンプルデータを読み込む）」をクリックすると、ダミーデータが入力され、アプリの見た目をすぐに確認できます。

## 📂 ファイル構成

- `index.html` - アプリケーションのセマンティックな構造とレイアウト。
- `styles.css` - デザイン・トークン、グラスモーフィズム効果、レスポンシブなレイアウトグリッド、アニメーションの設定。
- `app.js` - 状態管理、DOM操作、Chart.jsの統合、ローカルストレージのロジック。
- `print.css` - きれいなPDFや紙への印刷に特化したスタイルシート。

## 💾 データ管理

ブラウザを変更したり、データをバックアップしたい場合:
1. **Settings（設定）** タブに移動します。
2. **Export JSON（JSONをエクスポート）** をクリックして、財務データの完全なバックアップをダウンロードします。
3. 別のデバイスやブラウザで **Import JSON（JSONをインポート）** ボタンを使用すると、状態を復元できます。
